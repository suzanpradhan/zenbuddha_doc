
# Using the Button Component in React

In this guide, we'll explore how to use the `Button` component from the `react-to-ui` library. The `Button` component is versatile and can be customized to fit various use cases such as actions, links, and more.

### Example

``` jsx

import { Button } from 'react-to-ui';

function App() {
  return (
      <Button
        text="Submit"
        kind="success"
        onClick={() => alert('Button clicked!')}
      />

      <Button
        text="Cancel"
        buttonType="bordered"
        kind="danger"
        onClick={() => alert('Cancel clicked!')}
      />

      <Button
        text="Learn More"
        type="link"
        href="/learn-more"
        kind="secondary"
        tooltip="Click to learn more"
      />
  );
}

export default App;
