# Spinner Component

The `Spinner` component is a simple and lightweight loading indicator that can be used to show that a process is ongoing. It can be easily customized to fit the design of your application.


# Example

```jsx

import { Spinner } from 'react-to-ui';

function App() {
  return (
    <div >
      <Spinner />
    </div>
  );
}

export default App;
